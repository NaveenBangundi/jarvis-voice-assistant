\# 🤖 Jarvis Voice Assistant



A Python-based voice assistant inspired by Jarvis that can perform tasks using voice commands such as opening applications, searching the web, and automating basic system operations.



\---



\## 🚀 Features



\- 🎤 Voice command recognition  

\- 🌐 Web search functionality  

\- 💻 Open applications automatically  

\- 🔊 Text-to-speech responses  

\- ⚡ Fast and interactive UI  



\---



\## 🛠️ Tech Stack



\- Python  

\- HTML, CSS, JavaScript  

\- Speech Recognition  

\- Text-to-Speech (TTS)



\---



\## 📂 Project Structure



\- `jarvis.html` – Main interface  

\- `jarvis.css` – Styling  

\- `jarvis.js` – Logic and interactions  

\- Assets – Images and GIFs  



\---



\## 🎯 Purpose



This project was built to explore voice-based automation and create a simple AI assistant that improves productivity through hands-free interaction.
